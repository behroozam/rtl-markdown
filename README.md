# [RTL Markdown](http://md.behroozam.com)

A simple markdown editor, which supports right-to-left languages.

features currently support:
* support footnotes with GFM format
* support emoji

## Develop
```
npm install
npm run build

```
## Special Tags
Output text in left-to-right direction:

```
<%ltr%>
  Left To Right Text
<%endltr%>
```
